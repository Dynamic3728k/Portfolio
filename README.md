# Personal Portfolio Website

Welcome to my personal portfolio repository! This responsive website is designed to showcase my academic journey, technical skills, and projects as a **Computational Mathematics student at NIT Agartala**.

It serves as a central hub to view my work in **C++**, **Data Structures**, and **Web Development**.

## Live Demo
*(https://dynamic3728k.github.io/Portfolio/)*

## Features

* **Responsive Design:** Fully adaptable layout that looks great on desktops, tablets, and mobile devices (includes a custom hamburger menu for mobile).
* **Dark Mode UI:** A modern, sleek dark aesthetic using a custom color palette (`#080808` background with `#ff004f` accents).
* **Interactive Tabs:** "About Me" section utilizes JavaScript to toggle between Skills, Experience, and Education without reloading the page.
* **Project Showcase:** A grid layout displaying my recent projects (Amazon Clone, DFS Visualizer, etc.) with hover effects.
* **Contact Section:** Integrated links to my LinkedIn and GitHub, along with a styled contact form layout.
* **Smooth Scrolling:** Easy navigation between sections.

## Tech Stack

* **HTML5:** Semantic structure of the webpage.
* **CSS3:** Custom styling, Flexbox, CSS Grid, and Media Queries for responsiveness.
* **JavaScript:** Logic for the mobile side-menu and tab switching in the "About" section.
* **Icons:** FontAwesome for social media and interface icons.
* **Fonts:** Google Fonts ('Poppins').

## Project Sections

1.  **Header:** Introduction and navigation.
2.  **About:** Bio highlighting my background in Comp Math and C++, plus tabbed details.
3.  **Services:** Overview of my interests (Programming, Web Dev, Competitive Programming).
4.  **Portfolio:** Links to my GitHub projects.
5.  **Contact:** Social links and a message form.


##  How to Run Locally

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/Dynamic3728k/Portfolio.git](https://github.com/Dynamic3728k/Portfolio.git)
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd Portfolio
    ```
3.  **Open `index.html`:**
    Simply double-click the `index.html` file to open it in your default web browser.
